# MarkIt BI Dashboard Project

## Overview
MarkIt is a fictional consumer-to-consumer marketplace where users buy and sell previously owned items. This Business Intelligence project demonstrates the complete BI lifecycle: stakeholder requirements, project planning, SQL analysis, KPI design, dashboard strategy, and business recommendations.

## Business Problem
MarkIt wants to understand buyer behavior, seller behavior, search query patterns, listing performance, and completed sales so the company can improve user experience and increase successful sales.

## Skills Demonstrated
- Business Intelligence
- Stakeholder Requirement Analysis
- Project Planning
- SQL
- KPI Development
- Dashboard Design
- Data Storytelling

## Key KPIs
- Listings Posted
- Completed Sales
- Deleted Listings
- Search-to-Sale Ratio
- Average Days to Sale
- Total Revenue

## Folder Structure
- 01_Business_Documents: Stakeholder, project, and strategy documents
- 02_Dataset: Synthetic CSV data
- 03_SQL: Schema and analysis queries
- 04_Dashboard: Dashboard design plan and KPI definitions
- 05_Business_Insights: Executive summary and recommendations

## Tools Used
SQL, CSV/Excel, Power BI/Tableau planning, BI documentation frameworks.
