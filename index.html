Project Overview

MarkIt is a fictional online marketplace where users buy and sell previously owned items. The goal of this project is to demonstrate the complete Business Intelligence lifecycle: stakeholder requirements, project planning, strategy documentation, SQL analysis, KPI design, dashboard planning, and business recommendations.

Business Problem

Understand how buyers and sellers use the platform and identify what drives successful sales.

BI Objective

Create a dashboard that tracks listings, sales, deleted listings, searches, and user behavior.

Outcome

Support product, sales, and platform decisions using measurable KPIs and actionable insights.

SQL Analysis

The analysis uses SQL to measure marketplace performance and prepare data for dashboard reporting.

SELECT
    sale_date,
    COUNT(*) AS completed_sales,
    SUM(sale_amount) AS total_revenue
FROM sales
GROUP BY sale_date
ORDER BY sale_date;

Business Insights

Key Findings

  • Search behavior can indicate buyer purchase intent.
  • Deleted listings may reveal seller pain points.
  • Listing duration helps identify demand and pricing issues.
  • Category-level performance can guide marketing and product decisions.

Recommendations

  • Improve search functionality for broad search terms.
  • Monitor high-deletion categories for seller friction.
  • Promote categories with faster sales cycles.
  • Track search-to-sale ratio as a monthly business KPI.
